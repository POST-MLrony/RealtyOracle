**Проект "RealtyOracle", разработанный командой "POST MLrony", предлагает пользователям инновационный сервис для предсказания цен на недвижимость. Основываясь на широком спектре данных, включая местоположение, этажность, количество комнат и множество других ключевых характеристик, наша платформа использует передовые алгоритмы машинного обучения для генерации точных и надежных оценок стоимости недвижимости.**

## О проекте

RealtyOracle представляет собой инновационное решение, которое позволяет с высокой точностью прогнозировать цены на недвижимость. Мы достигаем высокой точности прогнозов благодаря использованию мощных алгоритмов машинного обучения. Наш подход предоставляет не только оценку стоимости, но и обеспечивает глубокую интерпретируемость полученных результатов. Наш сервис представляет прозрачный результат прогнозирования, благодаря чему пользователи могут видеть, какие факторы и в какой степени влияют на стоимость недвижимости.

## Основные характеристики

- **Цели проекта:**
Разработать инструмент, учитывающий большое количество факторов для более точного прогнозирования цен на недвижимость.
Данный сервис поможет улучшить качество прогноза и соответственно повысить точность прогнозирования, что актуально как для потребителей информации о рынке недвижимости, так и для специалистов в области недвижимости и инвестиций.
- **Технологии и инструменты:**
  - Бэкенд (API): Реализован на FastAPI.
  - Модель машинного обучения: Используется CatBoost.
  - Фронтенд (веб-интерфейс): Разработан с использованием HTML, CSS и JS.

## Установка

### Локальная установка

1. Склонируйте репозиторий:

   `git clone `

2. Перейдите в директорию проекта:

   `cd RealtyOracle/server`

3. Установите зависимости:

   `pip install -r requirements.txt`

4. Запустите приложение:

   `uvicorn app:app --reload`

   Приложение будет доступно по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

### Онлайн-версия

Вы также можете использовать [онлайн-версию](https://151b-77-238-135-243.ngrok-free.app) проекта для удобного взаимодействия с функциональностью сервиса.

### Веб-интерфейс
Веб-интерфейс проекта "RealtyOracle" предоставляет обширный функционал для удобного взаимодействия с сервисом. Вот краткое описание ключевых возможностей:

## 1. Главная страница
- Предоставляет основную информацию о проекте, команде разработчиков и контактную информацию.
- Пользователи могут сразу выбрать интересующий их город в шапке сайта или перейти на следующую страничку с помощью кнопки "Приступить".

## 2. Страница выбора города
- Пользователи могут выбрать город из списка, предложенного сервисом.
- Мы предусмотрели опцию для отправки обратной связи и сообщений разработчикам. Если ваш город отсутствует в списке или у вас есть предложения по улучшению работы сервиса, не стесняйтесь делиться вашими мыслями и идеями через форму обратной связи.

## 3. Страничка для прогноза цены на недвижимость в выбранном городе
- Пользователь может выбрать точку на карте самостоятельно или ввести адрес с помощью `API Яндекс.Карты`.
- Заполнение важных характеристик выбранного объекта.
- После тщательного заполнения всех необходимых сведений пользователем, наш сервис предоставляет экспертную оценку стоимости недвижимости в интересующем городе. Эта оценка базируется на инновационных алгоритмах машинного обучения, в частности, на использовании модели CatBoost — одного из наиболее прогрессивных инструментов в области предсказательной аналитики. Благодаря этому подходу, мы гарантируем высокую точность и надежность наших прогнозов, открывая новые возможности для наших пользователей в планировании их инвестиций в недвижимость.
- Для того чтобы наши пользователи могли не только видеть цифры, но и полностью понимать, откуда они взялись, мы внедрили технологию SHAP (SHapley Additive exPlanations). Этот метод раскрывает "за кулисами" процесс принятия решений нашей модели, предоставляя детализированный анализ влияния каждого фактора на конечную стоимость. Таким образом, мы не только даем вам точные цифры, но и предоставляем полную картину, делая интерпретацию результатов прозрачной и понятной.
